IX-Party Language Toggle Feature

The IX-Party website now includes language toggle functionality, allowing users to switch between Japanese and English.

Implementation Details

  • Added a language toggle button in the navigation bar
  • Created JavaScript with translations for all content
  • Added data attributes to elements that need translation
  • Implemented language switching functionality with localStorage for persistence

Key Features

  • Seamless toggling between Japanese and English
  • User language preference is remembered between visits
  • Complete translation of all site content
  • Minimal changes to the original site structure
